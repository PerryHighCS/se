println() and readLine()

The first thing I learned about in JavaScript is a method to print out text. I learned a simple method of outputting strings to the console, println().

println() prints out a line of text on the console then moves to the next line. A similar method, print() prints out text without moving to the next line.

readLine() prompts the user to type something in. When the User clicks OK, readLine() returns whatever they typed.
